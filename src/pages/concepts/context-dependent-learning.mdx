# Context-Dependent Learning

**Summary**: Research showing that learning and recall are significantly improved when they occur in the same context — and that emotional, situational, and linguistic context all shape how information is stored, retrieved, and applied.

**Sources**: CULTIVATING CLARITY.md

**Last updated**: 2026-04-14

---

## Key Findings (from Naumenko's "Memory" paper)

1. **State-dependent retrieval**: Information learned in one context is better recalled in the same context
2. **Emotional encoding**: Emotionally rich memories are stored faster and retained longer
3. **Dimensional reduction**: We reduce information dimensions for faster processing — not all details are stored
4. **Memory modification**: Retrieved memories are rarely identical to the initial experience; irrelevant details are removed over time
5. **Acoustic over visual**: We're more likely to confuse words that sound alike than look alike (true for deaf people with similar hand shapes)
6. **Rehearsal as internalization**: By rehearsing information, we "impose our point of view on what we observe" — turning implicit needs into explicit representation
7. **Spaced repetition**: Spaced learning across sessions is more efficient than massed learning
8. **Social learning limits**: Learning from others' mistakes is often overestimated

## Implications for Product Innovation

- **Design for context consistency**: Products used in specific contexts should be tested and refined in those same contexts
- **Leverage emotional anchors**: Features and experiences that create emotional moments will be better remembered and valued
- **Reduce cognitive load**: Don't present all information at once — match information density to the user's context
- **Support rehearsal**: Allow users to interact with, modify, and internalize information rather than passively consuming it
